Engram is a TypeScript monorepo (pnpm workspaces + Turborepo) that runs an
[MCP](https://modelcontextprotocol.io) memory server. It stores short-term and
long-term memories, embeds them for semantic recall, and exposes a suite of MCP
tools that any compatible agent (Claude Desktop, Claude Code, and others) can
call.
