Each guide solves one operational task, assuming a working deployment (start
with [Getting started](/docs/getting-started/) if you don't have one). For
the *why* behind these procedures, see [Architecture](/docs/architecture/).

## Run it in production

| Guide | Task |
| ----- | ---- |
| [Deploy to production](/docs/how-to/deploy-production/) | Build and run the hardened Docker image, compose file, health checks |
| [Back up and restore](/docs/how-to/backup-and-restore/) | pg_dump, Redis BGSAVE, Qdrant snapshots, retention, restore drills |
| [Enable observability](/docs/how-to/enable-observability/) | Prometheus metrics scraping and OpenTelemetry tracing |
| [Migrate lite to enterprise](/docs/how-to/profile-migration/) | Promote a profile-lite deployment to the full stack |

## Secure it

| Guide | Task |
